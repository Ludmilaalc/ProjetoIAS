# 🏗️ ProjetoIAS — Supervisão de Andaimes Suspensos

Aplicação em **Django** para gestão de inspeções de andaimes suspensos.  
Permite importar planilhas Excel, anexar imagens e gerar relatórios técnicos em PDF.
